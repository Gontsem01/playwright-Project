# Playwright Project

This project contains Playwright end-to-end tests for the Ndosi Simplified Automation website.

## Project structure

- `tests/` - Playwright test files
- `src/` - support files for test fixtures, page objects, utilities, and test data
- `playwright.config.ts` - Playwright configuration
- `package.json` - project dependencies and scripts

## Setup

```bash
npm install
```

## Run tests

```bash
npx playwright test
```

## Run a specific test

```bash
npx playwright test tests/Ndosi.spec.ts
```

## View report

```bash
npx playwright show-report
```
